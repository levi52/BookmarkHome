# BookmarkHome

> 🤖 **本项目由 AI（WorkBuddy）全程辅助生成与迭代开发**

将浏览器收藏夹**自动分类展示**为精美新标签页，告别混乱书签！

---

## 功能特性

- 📂 根节点展示 — 自动识别收藏夹栏、其他收藏夹、工作区，首页平铺显示
- 🏠 层级导航 — 点击根节点进入内容，面包屑导航随意跳转，Backspace 返回上一级
- 🔎 多搜索引擎 — 点击搜索图标切换收藏夹/Bing/Google/百度/Yandex，Enter 跳转外部搜索
- 🌙 深色/浅色主题 — 一键切换，偏好自动保存
- 📌 收藏夹实时同步 — 监听收藏夹变更，自动刷新
- ⚡ 快速跳转 — Popup 面板快速搜索 + 跳转
- 📦 折叠/展开所有 — 顶部工具栏一键批量折叠或展开
- ✏️ 重命名 & 删除 — 每个条目 hover 时显示操作按钮，即时同步到浏览器
- 🎯 分类过滤 — 顶部根节点 Tab 切换不同收藏夹分区
- 🎨 图标自动获取 — 自动显示每个网站的 Favicon，多重回退策略
- 🪶 零依赖 — 纯原生 JavaScript，无框架，轻量快速

---

## 安装方法

### Chrome / Edge / Brave

1. 打开浏览器，访问：`chrome://extensions/`（Edge 为 `edge://extensions/`）
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目文件夹
5. 安装完成！打开新标签页即可看到效果 🎉

## 项目结构

```
├── manifest.json            # 插件配置（Manifest V3）
├── icons/                 # 插件图标（16/32/48/128px）
├── newtab/
│   ├── newtab.html        # 新标签页主页
│   ├── newtab.css         # 样式文件（深色/浅色主题、动画）
│   └── newtab.js          # 主逻辑（根节点/导航/搜索/操作按钮）
├── popup/
│   ├── popup.html         # Popup 弹窗
│   └── popup.js           # Popup 逻辑（最近书签/搜索）
├── background/
│   └── background.js      # Service Worker（数据处理/收藏夹API/剪切板）
└── README.md
```

## 使用说明

| 功能 | 操作 |
|------|------|
| 打开主页 | 新建标签页自动显示 |
| 搜索书签 | 顶部搜索框 / 按 `/` 或 `f` 键聚焦 |
| 退出搜索 | 按 `Esc` 键 |
| 进入根节点 | 首页点击根节点卡片 |
| 返回首页 | 点击面包屑「🏠 首页」/ 根节点内容页按 `Backspace` |
| 进入子文件夹 | 点击子文件夹按钮 |
| 重命名条目 | Hover 卡片/条目 → 点击 ✏️ 按钮 |
| 删除条目 | Hover 卡片/条目 → 点击 🗑️ 按钮 |
| 折叠单个分类 | 点击分类卡片标题栏 |
| 展开/折叠全部 | 顶部工具栏按钮 |
| 切换主题 | 顶部月亮/太阳图标 |
| 刷新数据 | 顶部刷新图标 |
| 快速搜索 | 点击浏览器工具栏插件图标 |

## 技术栈

- **Manifest V3** — Chrome Extensions 最新规范
- **原生 JavaScript** — 无任何框架依赖，轻量快速
- **Chrome Bookmarks API** — 读取和管理收藏夹数据
- **Chrome Storage API** — 保存用户偏好设置
- **Chrome Favicon API** — 获取网站图标
